# Swiss Theme

A professional neutral theme with balanced grays. Perfect for business and corporate environments.

## Color Palette

- **Primary**: `#4B5563` (Gray-600)
- **Secondary**: `#9CA3AF` (Gray-400)
- **Accent**: `#6B7280` (Gray-500)
- **Light**: `#F9FAFB` (Gray-50)
- **Dark**: `#111827` (Gray-900)

## Features

- Professional, neutral appearance
- Balanced gray color palette
- Corporate-friendly
- Excellent for business environments
- Clean, understated design
