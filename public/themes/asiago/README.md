# Asiago Theme

A sophisticated dark theme with cream and gold accents. This is the original theme for Cheesebox.

## Color Palette

- **Primary**: `#181717` (Deep black-brown)
- **Secondary**: `#FAEACB` (Cream)
- **Accent**: `#F5BE4B` (Gold)
- **Light**: `#FAEACB` (Cream)
- **Dark**: `#181717` (Deep black-brown)

## Features

- Dark, professional appearance
- High contrast for readability
- Warm color accents
- Status colors for success, warning, danger, and info states
- Full support for all button variants
- Gradient support for special components

## Status Colors

- **Success**: Green (`#10B981`)
- **Warning**: Amber (`#F59E0B`)
- **Danger**: Red (`#EF4444`)
- **Info**: Blue (`#3B82F6`)

## Installation

This theme is built into Cheesebox. To use it:

1. Navigate to Settings
2. Select "Asiago Dark" from the theme switcher
3. Click "Apply Theme"

## Development

This theme follows the Cheesebox theme structure:

- `theme.ts` - Theme configuration object
- `styles.css` - CSS variables and utilities
- `package.json` - Theme metadata
- `preview.png` - Theme preview image
- `README.md` - Documentation (this file)

All styles are scoped to `[data-theme="asiago"]` for hot-swapping support.
