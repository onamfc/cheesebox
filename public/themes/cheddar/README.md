# Cheddar Theme

A warm and energetic theme with orange and yellow tones. Perfect for a vibrant, creative workspace.

## Color Palette

- **Primary**: `#F59E0B` (Amber-500)
- **Secondary**: `#FCD34D` (Amber-300)
- **Accent**: `#FBBF24` (Amber-400)
- **Light**: `#FEF3C7` (Amber-100)
- **Dark**: `#78350F` (Amber-900)

## Features

- Bright, energetic appearance
- Warm color palette
- High contrast for readability
- Perfect for creative work
- Full support for all button variants

## Installation

This theme is built into Cheesebox. To use it:

1. Navigate to Settings
2. Select "Cheddar Bright" from the theme switcher
3. Click to preview, then confirm to apply
